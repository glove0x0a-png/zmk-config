## ✅ Git操作手順（ZMKサブモジュール追加＆Push）

### 🧼 ① 既存の `app` をGitから削除（登録だけ消す）
git rm --cached app

---

### 🧹 ② ローカルの `app` フォルダを削除（必要なら）
rmdir /s /q app
※Windowsのエクスプローラーで削除してもOK！

---

### 🔁 ③ ZMKを `app` に再追加（サブモジュール）
git submodule add https://github.com/zmkfirmware/zmk app

---

### 📦 ④ ZMKの中身を取得
git submodule update --init --recursive

---

### 💾 ⑤ Gitに登録してコミット
git add .gitmodules app
git commit -m "Add ZMK firmware as submodule"

---

### 🚀 ⑥ GitHubにPush！
git push

---

これで水源（ZMK本体）がGitHubに流れて、GitHub Actionsがビルドを始められるよ～！🌈  
うまくいったら、`.uf2` ファイルが生成されるはず！
